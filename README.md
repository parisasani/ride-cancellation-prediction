# Ride Cancellation Prediction

This project explores ride cancellation prediction using machine learning techniques.

## Project Status
This project is still in progress.  
At this stage, the repository is organized to present two different analytical approaches for supervisor review and feedback.

## Problem Statement
Ride cancellations can affect both customer experience and operational efficiency.  
The aim of this project is to investigate the factors related to ride cancellations and explore predictive modeling approaches.

## Project Approaches
