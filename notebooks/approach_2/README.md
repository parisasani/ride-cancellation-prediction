# Approach 2 (Enhanced Model with Feature Engineering)

This approach extends the baseline model by incorporating feature engineering and deeper analytical insights.

## Objective
The goal of this approach is to improve prediction performance by creating more meaningful features and gaining deeper insights into ride cancellation behavior.

## Methodology
This approach includes the following steps:

1. Data Cleaning & Audit  
The dataset is carefully cleaned and audited to ensure data quality and reliability.

2. Feature Engineering  
New features are created based on existing variables such as waiting time, booking attributes, and ride characteristics to improve model performance.

3. Exploratory Data Analysis & Business Insights  
EDA is extended to extract deeper insights and understand key drivers of ride cancellations from a business perspective.

4. Modeling  
Machine learning models are applied using improved features to enhance predictive performance.

## Description
Compared to Approach 1, this approach introduces feature engineering and more advanced analysis.  
It aims to better capture the underlying patterns in the data and improve the accuracy of ride cancellation predictions.
